主要学习与记录 **`TypeScript`** 的语法特性及用法示例。

### Run
```shell
1. git clone https://github.com/jwchan1996/TSstudy.git
2. cd TSstudy  
3. npm install  
4. npm run dev  
5. 访问 http://127.0.0.1:1996  
```
